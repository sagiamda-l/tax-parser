# 📑 Tax-Parser: 통합 연말정산 및 지출 내역 분석 시스템

폐쇄망 환경(Offline Environment)에서 보안을 유지하며 개인 지출 내역을 체계적으로 관리하기 위한 풀스택 애플리케이션입니다.

---

## 🎯 프로젝트 원칙 (Ground Rules)
1. **데이터 무결성**: 모든 내역은 업로드된 파일(UploadFileRecord)에 귀속되며, 파일 삭제 시 관련 내역은 연쇄 삭제(Cascade)됩니다.
2. **고유 식별자**: 모든 파일 PK는 시각적 추적이 가능하도록 다음의 수식을 따릅니다.
   - $PK = CurrentTimestamp(17\text{ digits}) + Random(5\text{ digits})$
3. **사용자 중심 필터링**: 모든 데이터 조회는 '대상 년도'를 기준으로 하며, 유동적인 콤보박스 필터를 제공합니다.

---

## 🛠 주요 기능 상세

### 1. 파일 관리 및 업로드 워크플로우
- **검증**: 파일 업로드 시 서버는 파일명 중복을 체크합니다.
- **분기**: 
  - 중복 발생 시: 클라이언트에게 `409 Conflict` 반환 -> 팝업을 통해 '덮어쓰기(Overwrite)' 확인.
  - 신규 파일: 즉시 파싱 및 저장.
- **이력 저장**: PK, 파일명, 업로드 시간, 이용자, 문서종류, 대상년도 정보를 기록합니다.

### 2. 상세 내역 그리드 (Data Grid)
- **추가 컬럼**: [문서명], [사용자] 정보를 포함합니다.
- **태그 시스템**: 
  - 시각화: 태그별 아이콘(🍴, 🚌 등) 및 배경색 적용.
  - 편집: 그리드 내 select 박스를 통해 태그 수정 후 '일괄 저장' 가능.
- **동적 필터**: 현재 화면에 표시된 데이터의 유저명과 파일명을 추출하여 필터 콤보박스 아이템을 생성합니다.

### 3. 데이터 시각화
- **단위**: Y축 금액을 '만 원' 단위로 변환하여 가독성 확보.
- **디자인**: Pretendard 폰트 적용 및 레이아웃 최적화.
